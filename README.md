# Facebook-Message-Delete
This program is automated to delete the facebook meessage.

You have to download chromedriver.exe and give a path name in the program so that you can work with it.
link to download:
https://sites.google.com/a/chromium.org/chromedriver/downloads

You have to download geckodriver.exe for firefox.
Link to download is: 
https://github.com/mozilla/geckodriver
